# Frontend Framework Standards

Gold-standard conventions for frontend frameworks across the Ummard/Simon ecosystem.
Each subfolder holds a `README.md` with naming conventions, project structure, tooling,
and best practices for that framework.

Each folder holds **thin conventions** (its `README.md`) plus a nested **`architecture/` deep-dive** — a full production application-architecture standard with a rejected-anti-patterns section.
